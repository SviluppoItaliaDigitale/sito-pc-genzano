Checklist evacuazione paziente non autosufficiente

Per OSS / infermieri — 1 scheda per ogni paziente trasferito
📋 Spunta tutto prima di partire. 5 minuti di checklist ora = ore di problemi risolti. La cartella va con il paziente, mai separata.
Paziente: ____________________________ Stanza: ____ Data trasferimento: ____ / ____ / ____
Operatore responsabile (firma): ____________________________

📦 BAG DI TRASFERIMENTO 72H

  • Cartella clinica completa (cartacea + USB se disponibile)
  • Carta d'identità RSA per emergenza (Scheda 1)
  • Documento d'identità + tessera sanitaria (originale)
  • Terapia farmacologica per minimo 72h, in scatola etichettata col nome
  • Insulina / anticoagulanti / antiparkinson — verifiche specifiche
  • Diario delle ultime 24h (vital signs, somministrazioni, eventi)
  • Indumenti per minimo 3 giorni
  • Pannoloni / presidi continenza per 72h

🦯 AUSILI PERSONALI

  • Occhiali (controllare presenza, eventuale custodia)
  • Apparecchio acustico (con pile di scorta)
  • Dentiera (con custodia)
  • Deambulatore / bastoni / carrozzina
  • Materasso anti-decubito (se in uso)

❤️ COMFORT / DIGNITÀ

  • Oggetto di conforto personale (foto familiare, rosario, coperta)
  • Carta del comfort sensoriale (Scheda 18) se demenza grave
  • Etichetta visibile sul paziente con nome, cognome, data di nascita
  • Indicatore "non separare da [coniuge/convivente]" se presente

🩺 PRESIDI MEDICI (se applicabili)

  • PEG: kit di gestione + boli per 72h
  • Catetere vescicale: sostituzione + sacche per 72h
  • Ossigeno: bombola portatile + recupero al ricovero ricevente
  • Stomia: presidi per 72h
  • Tracheostomia: aspiratore portatile + sondini

📞 COMUNICAZIONE

  • Familiare di riferimento informato (chi/quando/da chi: indicare in cartella)
  • Struttura ricevente avvisata (orario stimato arrivo)
  • Trasporto disposto (ambulanza / mezzo PC / mezzo aziendale: indicare)
