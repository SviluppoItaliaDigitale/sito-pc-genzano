## 4. Come trasformare una fonte in un articolo (procedura)

### Passo 1 — Trova la notizia
Consulta le fonti della sezione 1. Esempi tipici:
- Il DPC pubblica un bollettino di criticita per il Lazio
- La Regione annuncia un bando per i volontari
- L'INGV registra una scossa avvertita nel Lazio
- Il Ministero della Salute attiva il bollettino calore

### Passo 2 — Adattala per Genzano
Non copiare: **rielabora** il contenuto con queste regole:
- Scrivi in prima persona plurale ("Il nostro territorio...", "I nostri volontari...")
- Contestualizza per Genzano: "Per il territorio di Genzano e dei Castelli Romani..."
- Aggiungi i consigli pratici per i cittadini
- Cita sempre la fonte originale con link
- Aggiungi il riferimento ai numeri di emergenza se pertinente

### Passo 3 — Pubblica
Usa il comando:
```bash
hugo new comunicazioni/AAAA-MM-GG-titolo-articolo.md
```
Oppure chiedi a Claude Code di scrivere l'articolo.

### Esempio concreto

**Fonte**: Il Centro Funzionale Regionale emette allerta gialla per temporali sul Lazio.

**Articolo adattato**:
- Titolo: "Allerta gialla per temporali: consigli per i cittadini di Genzano"
- Badge: Allerta
- Contenuto: spiegare cosa significa allerta gialla, quali zone sono interessate, cosa fare prima/durante/dopo, numeri da chiamare, link al bollettino ufficiale

## 6. Suggerimento operativo

Dedica **30 minuti a settimana** al sito:
- Lunedi: controlla le fonti (10 min)
- Se c'e qualcosa di rilevante, scrivi una bozza o chiedi a Claude Code di scriverla (20 min)
- Se non c'e nulla di urgente, lavora su un contenuto "sempre valido" dalla lista della sezione 3
